Box Model

A box model is a box that wraps every HTML element.It contains multiple properties including borders, margin, padding and the content itself.It is used to create design and layout web pages. It is used as a toolkit for customizing the layout of different elements.

Properties of a box model.

  • Content
  • Borders
  • Margin
  • Padding
  • Box-sizing
  • Background clip
  • Display

Border area-is the area between the box's padding and margin.
Margin area-is the space between the border and margin.
Padding area-It is the space around the content area and within the border box.

Calculating the total width of a paragraph.

Width:400px Padding:20px Margin:20px Border:2px Formulae;
Total width=width+padding(left,right)+border(left,right)+margin(left,right)
width=400+20+20+2+2+20+20
width=484px.
